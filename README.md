# def-disconnect-tool
OpenSCAD script to generate a DEF supply line disconnect tool . The default parameters are set to render a 1 3/8" gap and tool thickness of 1/4"

## PreRequisites
* A  3D Printer
* OpenScad (if you want to tweak the script and / or design) -  http://openscad.org/

## Usage
Pre-built STL files available on Thingiverse at https://www.thingiverse.com/thing:7323716

## Customize
To change parameters on the fly, simply open the Customizer panel in OpenSCAD (usually located on the right side of the screen). If it isn't visible, go to the View menu and uncheck Hide Customizer. This interface turns your script variables into intuitive sliders and input boxes, allowing you to adjust dimensions like opening_width or plate_thickness and see the 3D model update instantly without ever having to touch the underlying code.

Then, export as an STL in OpenScad
